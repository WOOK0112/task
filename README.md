### 애플리케이션 실행 플로우
***엔드포인트 호출의 상세내용은 API명세서 참고바람
1. IDE를 이용하여 애플리케이션을 실행
2. API개발 테스트툴(ex. POSTMAN)을 사용하여 애플리케이션 서버와 통신
3. 회원가입 진행
4. 가입한 계정으로 로그인
5. 로그인 후 게시글 작성, 조회, 수정, 삭제 실행 가능
   - 게시글을 작성한 계정에 한하여  게시글 작성, 수정, 삭제 가능
   - 게시글 단일 조회, 목록 조회는 로그인 없이 가능
